# Семантический поиск среди русских пословиц с интерактивной визуализацией

Проект выполнен индивидуально в рамках учебного курса по компьютерной лингвистике

---

## 1. Описание идеи

Проект решает задачу контекстного поиска русских пословиц, преодолевая лексические ограничения стандартного текстового поиска. Система использует дообученную языковую модель для сопоставления произвольного текстового описания с метафорическим смыслом пословицы, находя наиболее близкие варианты и визуализируя положение запроса на интерактивной двумерной смысловой карте. Инструмент ориентирован на исследователей фольклора, лингвистов и преподавателей русского языка.

---

## 2. Задачи проекта

В рамках работы решен комплекс лингвистических и технических задач: осуществлен сбор и очистка базы пословиц через API, проведено контрастивное дообучение модели LaBSE на корпусе парафраз, построен векторный индекс FAISS для быстрого семантического поиска, выполнена кластеризация методом K-Means и t-SNE проекция для построения карты смысловых групп, а также разработан единый веб-интерфейс на Gradio с блокировкой ручного изменения масштаба карты.

---

## 3. Как работает система

Сначала модель LaBSE проходит дообучение методом контрастивного обучения на корпусе парафраз inkoziev/paraphrases, обучаясь сопоставлять синонимичные конструкции. Параллельно база пословиц скачивается через API Викицитатника и очищается. Векторы пословиц проецируются в 2D через t-SNE и разбиваются на восемь кластеров методом K-Means с сохранением пропорций осей. При вводе запроса его координаты на карте рассчитываются как средневзвешенное значение координат пяти ближайших соседей, позиционируя маркер в виде звезды в центре нужного семантического облака. Интерфейс на Gradio изолирует события поиска от выпадающего списка во избежание сброса масштаба карты, а ручной выбор конкретной группы генерирует под картой случайную подборка из пяти пословиц этой категории.

---

## 4. Инструкция по запуску

Проект запускается в Jupyter Notebook на любой платформе с подключенным графическим процессором (рекомендуется Kaggle GPU). Для работы достаточно последовательно выполнить все ячейки ноутбука и в конце перейти по сгенерированной ссылке веб-интерфейса Gradio.

---

## 5. Возможные направления развития

В дальнейшем систему можно развить за счет расширения базы пословиц академическими словарями (например, сборником В. И. Даля), добавления кросс-языкового сопоставления с иностранными аналогами и внедрения фильтрации по синтаксической структуре предложений.
